# Godot-Expo-Icons

A simple Godot 4.5 addon that allows the usage of all Expo Vector icons, as of current to 2026.

Source font files pulled from https://github.com/expo/vector-icons/tree/master

## Contains

AntDesign,
Entypo,
EvilIcons,
Feather,
FontAwesome,
FontAwesome5,
FontAwesome6,
Fontisto,
Foundation,
Ionicons,
MaterialCommunityIcons,
MaterialIcons,
Octicons,
SimpleLineIcons,
Zocial

## Customizability

By extending from the base `IconBase` class, you can easily create your own icon nodes using a new font file.
